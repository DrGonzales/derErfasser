# Description
eine PWA app für das erfassen von Messdaten für elektriche Sicherheitsprügungen

# Features
- App muss offline arbeiten. Werder Daten von Appanteile dürfen aus dem Netz geladen werden.
- Daten werden in der IndexDB gespeichert, import und export über Zipfiles 
- In der IndexDB stehen Json Daten als auch Bilder


# Requirements
- Node.js >= 14.x
- npm
- svelete
